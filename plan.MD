### Features
- Display list of issue
- Open individual issue
- Bookmark issue

### Data Entities
- Users
- Issues
- Bookmarks
- Users bookmark issues

### API routes
- GET /issues (github API)
- GET /issues/:id (github API)
- GET /bookmarks
- POST /bookmarks
- DELETE /bookmarks/:id

### Tech stack
- React frontend
- Node.js backend
- supabase user auth and DB
- Github Issues API
- DeepWiki (optional)
